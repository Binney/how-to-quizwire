for

Let's say you wanted to write down all the numbers from 1 to 10 on a web page. You could just write them out by hand... but that'd take a while, and what if you wanted to count to 100? It would take ages! Instead, you can get the computer to do it for you by using a for loop. A for loop looks like this:

for (i = 1; i <= 10; i+=1) { console.log(i); }

When you execute that, you'll get something which looks like this:

Pretty handy! Let's look at what's going on here more closely. The for loop is made up of 3 parts. 1) i=1 This declares a new variable called i, and sets it to 1. If we changed it to 5, we'd get something different:

for (i = 5; i <= 10; i+=1) { console.log(i); }

2) i <= 10 This is the condition for when our loop is going to stop. Each time the loop starts, we first check if this condition is true - if it's not, we stop looping. In this case, we're telling it to stop when i gets to 10.

for (i = 1; i <= 5; i+=1) { console.log(i); }

3) i+=1 This means "increase i by 1". Each time our loop finishes, we add 1 to i, check if our stop condition has happened yet, and then carry out the loop again.

for (i = 1; i <= 10; i+=2) { console.log(i); }

Ok, time for an exercise! We're going to make a duplication machine. It's going to take a picture on the page, and duplicate it a certain number of times. First, here's the image:

Save it into your Work Experience folder. Then add this HTML to your web page:

<div id="target"> <img src="butterfly.png"> </div>

We also need a box for users to type in how many butterflies they want:

<inupt type="number" id="copies">

(Tip! If you put type="number", then the user will only be able to type numbers into that box, no letters.) And we want a button we will use to call our duplication function:

<button onclick="duplicate()">Click to duplicate</button>

So it's time to write our function. Let's start with the basics:

function duplicate() { }

We need to get the inner HTML in our image, and the number of duplicates requested:

function duplicate() { var image = document.getElementById("target").innerHTML; var copies = document.getElementById("copies").value; }

All that remains now is to write our for loop. We want to start at i = 1, and end at i <= copies, increasing i by 1 each time.

for (i = 1; i < copies; i++) { }

Excellent: now we just need to make one copy of the butterfly each time.

for (i = 1; i < copies; i++) { document.getElementById("result").innerHTML += image; }

So the whole function looks like this:

function duplicate() { var image = document.getElementById("target").innerHTML; var copies = document.getElementById("copies").value; for (i = 1; i <= copies; i++) { document.getElementById("result").innerHTML += image; } }
