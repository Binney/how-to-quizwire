Welcome to the course! Let's start by learning about HTML.

HTML is the language that websites are written in. You can see some examples using Chrome's Developer Tools.

How to access Developer Tools

Right click on a web page and click Inspect (it's at the bottom).

You should see a window pop up at the side of the screen. This shows you the pure HTML, which your browser (i.e. Chrome, Edge, Safari, etc) will render into a web page you can read. It looks a little different to the web page itself but can you see the elements of the page showing up in HTML?

How to change HTML with Developer Tools

Find the HTML you want to change in the Elements tab. Double click and then start typing. Your new HTML will replace the old version.

Watch out! Your changes don't get saved. If you refresh the page, they will be gone! Developer Tools is only for testing.

Tags

You'll have noticed when you were inspecting this web page that there are loads of angle brackets (< and >) in the HTML. These are used to build what we call tags.

A tag might represent a section of the page, a picture, a link, or something else. You can tell what sort of tag it is by reading the word or letter between the angle brackets. For example, this paragraph is enclosed by <p> tags, and p stands for paragraph.

Important facts about tags

  1. For every open angle bracket there is an equal and opposite close angle bracket. If you don't close every tag, watch out! Your webpage won't work.
  2. Similarly, if you open a tag you have to close it! For instance, if you write a <p> you have to also write a </p>. The slash means you're closing it.
  3. Your web page will still work if you don't close all of your tags, but some things will break. However, if you don't close your brackets it will properly break, like properly. Try deleting a bracket or two with Developer Tools and you'll see what I mean.
