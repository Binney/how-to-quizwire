Making your web site interactive with Javascript

Now we're going to write some real code! Javascript is a super useful language that almost every computer in the world can "speak": it's used to interact with a web page.

How to add a Javascript script

First, you create a new Javascript file. Then you tell your web page to run that Javascript.

Create an empty new file and call it script.js. Save it in the same folder as your lesson4.html.

This is a Javascript file! We'll be writing Javascript and then your web browser will run it. Try writing this in your script.js:

alert("Hello, world!");

Now we have to tell your web page to run this Javascript like a script. You do this with the special "script" tag:

<script type="text/javascript" src="script.js"></script>

Try loading up the web page in your browser. You should see a popup message, nicely done!

What do you use Javascript for?

You can use Javascript to replace part of a web page. For instance, using document.getElementById("myDiv")

Create a button, and use the onClick handler to connect it to a function.

What's a function?

You might have come across the word "function" in maths. There, it means a process where you can put a number in one end and get a number out the other end.

Well, in computing functions are much the same. You send inputs in one end, and outputs come out the other end. Here's an example of a Javascript function which multiplies two numbers together:

function multiply(x, y) { return x * y; }
Copy this function into your Javascript file and save it.

Let's examine it more closely. We start with "function", because guess what, it's a function. "multiply" is the name of the function. Then, we have (x, y). These are our inputs, which in computing we call arguments. Finally we use the word return for our outputs.

Open your HTML in the browser, and go to the Javascript console.

You can call the function like this:

console.log(multiply(2, 3));

Try it out! Go to the console and type out "multiply(2,3)" and hopefully you'll see 6.
