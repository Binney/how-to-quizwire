Lesson 7: Making a Quiz Website

The time has come for us to start work on our main project: we're going to make a quiz game!

Create a new empty file and call it quiz.html. Then, create another and call it quiz.js.

For this project, we're going to use a tool called JQuery. It'll make Javascript easier to write.

Visit the JQuery website and click "Download the compressed, production JQuery". Save that file to your work experience folder.

If you open up that file, you'll see it's very hard to read! But JQuery is just another Javascript file - like the ones you've been writing. Now we'll include it in our HTML:

<script type="text/javascript" src="jquery-3.2.1.min.js"></script>

(If the numbers at the end of your filename are different, make the src match the version in your folder.) And also include your own Javascript file, so you'll get something like this:

<script type="text/javascript" src="jquery-3.2.1.min.js"></script> <script type="text/javascript" src="quiz.js"></script> </body> </html>
